# Gurmukhi Voice Search

Speak, or type, a line of Gurbani and find it in Sri Guru Granth Sahib. A lightweight web
rebuild (2026) of an idea first prototyped in 2024.

## This version

A fast, deployable MVP focused on **phonetic matching**: it transliterates input to a phonetic
form and matches it against the SGGS word corpus, with typed search as a fallback. Built with
Next.js and TypeScript.

## History

This is **v2**, a fresh and lighter take. The original 2024 research prototype (FastAPI with an
ML pipeline using wav2vec2 and sentence-transformers) lives at
**[gurmukhi-voice-search-prototype](https://github.com/thehimmat/gurmukhi-voice-search-prototype)**.
v2 is a work in progress.
